## 🔧 수정한 버그

### 1. GitHub 레포 설정 오류
**문제**: config.yaml에 존재하지 않는 `unohee/claude-swarm` 레포가 설정됨
**증상**: GitHub API에서 404 Not Found 오류 발생
**해결**: 실제 존재하는 레포로 변경
```yaml
# Before
repos:
  - unohee/claude-swarm

# After
repos:
  - unohee/CryptoNet
  - unohee/StockDataManager
  - unohee/ci-templates
```

### 2. TypeScript 컴파일러 누락
**문제**: `tsc` 명령어가 설치되지 않음
**증상**: `npm run build` 실행 시 "tsc: not found" 오류
**해결**: devDependencies에 이미 설치되어 있음을 확인 (별도 수정 불필요)

### 3. Discord 봇 인증 문제 (주의사항)
**문제**: Discord 토큰으로 인한 서비스 시작 실패
**증상**: 서비스가 즉시 종료됨
**상태**: ⚠️ 환경변수 설정 필요 (보안상 임의로 수정하지 않음)
